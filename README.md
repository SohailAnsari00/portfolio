# Sohail Ansari Portfolio

Personal portfolio website for **Sohail Ansari**, focused on data analytics, data engineering, AI/ML, and full-stack development experience.

## Overview

This is a responsive static portfolio built with HTML, CSS, and vanilla JavaScript. It presents profile information, experience, projects, skills, certifications, contact links, and a downloadable resume.

## Features

- Responsive portfolio layout for desktop and mobile
- Fixed navigation with mobile menu support
- Hero section with animated text styling
- About, experience, projects, skills, certifications, and contact sections
- Downloadable resume PDF
- Custom cursor and smooth reveal animations
- Google Fonts integration
- No build tools or framework required

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
- Google Fonts

## Getting Started

Clone the repository:

```bash
git clone https://github.com/SohailAnsari00/your-repo-name.git
```

Open the project folder:

```bash
cd your-repo-name
```

Run the site by opening `index.html` directly in your browser.

You can also use a local development server, for example with VS Code Live Server.

## Customization

- Update portfolio content in `index.html`
- Modify styling, layout, colors, and responsive behavior in `style.css`
- Replace `sohail_ansari_resume.pdf` with an updated resume file
- Update project links, contact links, and social profiles inside `index.html`

## Deployment

This project can be deployed easily on:

- GitHub Pages
- Netlify
- Vercel
- Cloudflare Pages

For GitHub Pages, enable Pages from the repository settings and select the branch containing `index.html`.
